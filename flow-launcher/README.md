# Flow Launcher

Cross-platform launcher for Windows.

## Setup

1. Flow Launcher is installed via Scoop during `bootstrap.ps1`
2. Settings are stored in `%APPDATA%\FlowLauncher\Settings\`
3. To sync settings, export from Flow Launcher → Settings → General → Export
4. Save the exported file here and restore on new machines

## Key Settings

- Hotkey: `Alt+Space` (macOS Spotlight-style)
- Theme matches Catppuccin aesthetic
- Plugins: Calculator, Everything, WebSearch, Programs
